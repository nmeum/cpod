=head1 NAME

cpod-import - OPML import for cpod(1)

=head1 SYNOPSIS

B<cpod> I<FILE>

=head1 DESCRIPTION

cpod-import imports your subscriptions from the specfied OPML I<FILE>.
After parsing the file it writes each outline to the cpod(1) urlfile.
After importing your feeds you need to manually invoke cpod(1) to fetch
them. If you want to export an existing OPML file take a look at
cpod-export(1).

=head1 SEE ALSO

cpod(1), cpod-export(1), cpod-import(1)
